/reservations

/reservations get

get

fetches current (and future) reservations made by clients and information regarding these clients

get /reservations

fetches current (and future) reservations made by clients and information regarding these clients

  • Request
  • Response

Query Parameters

  • currentOnly: (boolean)

    get reservations that are currently underway or all reservations (including ones that begin in the future)

  • roomID: (integer)

    get reservations connected with room with ID equal to roomID parameter

  • pageNumber: (integer - default: 1)

    Number of page to show results

    Example:

    20
  • pageSize: (integer - default: 10)

    Number of show results

    Example:

    20

HTTP status code 200

Body

Media type: application/json

Type: array of object

Items: reservationObject

  • reservation: required(object)
    • reservationID: required(integer)
    • hotelRoomNumber: required(string)
    • offerID: required(integer)
    • fromTime: required(datetime)
    • toTime: required(datetime)
    • childrenCount: required(integer)
    • adultsCount: required(integer)
  • client: required(object)
    • clientID: required(integer)
    • name: required(string)
    • surname: required(string)

HTTP status code 403

Room with ID equal to roomID does not belong to the hotel

HTTP status code 404

Body

Media type: application/json

Type: any

Example:

{
  "error": "Room with ID equal to roomID parameter does not exist"
}
